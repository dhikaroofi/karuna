# Background Music Setup

## Audio File Location

Place your background music file at:
```
public/music/background.mp3
```

## Supported Formats

- MP3 (recommended)
- WAV
- OGG

## Recommendations

1. **File Size**: Keep under 5MB for faster loading
2. **Duration**: 2-4 minutes (it will loop automatically)
3. **Volume**: The player is set to 30% volume by default
4. **Format**: MP3 works on all browsers
